# Add4x

Modern multi-tenant SaaS platform for restaurant operations.

## Core Services

| Service | Description |
|---------|-------------|
| [restaurant-backend](https://github.com/Add4x/restaurant-backend) | API & business logic (Node.js/Fastify) |
| [restaurant-admin](https://github.com/Add4x/restaurant-admin) | Administrative dashboard (Next.js) |
| [restaurant-ui](https://github.com/Add4x/restaurant-ui) | Customer-facing interface (Next.js) |
| [restaurant-kds](https://github.com/Add4x/restaurant-kds) | Kitchen Display System (React Native/Expo) |
| [restaurant-pos](https://github.com/Add4x/restaurant-pos) | Point of Sale terminal |
